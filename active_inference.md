# Active Inference in the Quantum Logical Framework

The universe functions as an **information ecology**. In such a system, anything that *creates* information has a natural evolutionary advantage, while anything that merely *consumes* it without creating more tends to disappear.

This is why active inference — by actively creating new information through the reduction of uncertainty — has a fundamental evolutionary edge. It explains why we observe order emerging in the universe rather than disorder.

## What the Python Demo Demonstrates

Our `active_inference_demo.py` shows a simple agent using Zero Free Action minimization to select among possible actions. The output demonstrates possibilistic agency: instead of following a fixed deterministic path, the agent evaluates possible futures and chooses the one that minimizes surprise.

This mirrors how agency works in the larger framework — selecting among timeless logical possibilities rather than determining them. The deterministic motor runs *after* the selection is made.

This work directly builds on Karl Friston's "The Physics of Sentience."
