## Abstract

Universal Relativity is the proposed unification of quantum physics, special relativity (SR), general relativity (GR), cosmology, and string-like extended structure under a single quantum-logical ontology.

In this framework the universe is **possibilist**: all admissible logical histories exist *a priori*; only histories that achieve **Zero Free Action (ZFA = 0)** are realized as physical events. Spacetime, light speed, mass, gravity, quantum measurement, the cosmological term, and the apparent age of the observable universe emerge from ZFA closures in an 8-twist algebra:

```text
^ v < > / \ + -
```

Einstein postulated a constant speed of light and the equivalence principle. Universal Relativity postulates ZFA and treats constant local `c`, local Lorentz invariance, and the equivalence principle as consequences of event closure. The theory is intended to be covariant, singularity-free, and computable.

The universe does not begin from a physical singularity. Instead, what observers call the **age of the universe** is the accumulated proper time synthesized by the realized ZFA event history of our observable cosmos. The repository update in `AgeOfUniverse.md` expresses this as an effective age of approximately **13.8 billion years**, derived from the observed frequency distribution of ZFA events rather than imposed by a Big-Bang singularity.

The framework is implemented in the open-source **Quantum Logical Framework (QLF)** repository. It includes executable Python engines, RhoQuCalc process calculus, and Lean4 formalizations/proof sketches.

## 7. Relation to String Theory

Universal Relativity includes a natural bridge to string theory, but it does not treat strings as fundamental objects in a pre-existing spacetime background.

String theory begins with extended one-dimensional objects whose vibrational modes appear as particles. Universal Relativity begins with **history strings** in the 8-twist algebra. A particle-like state is a stable ZFA-closed pattern of logical action. A string-like object is therefore not an assumed primitive. It is a higher-order process made from many ZFA closures.

The relationship can be summarized as follows:

| Question | String Theory | Universal Relativity / QLF |
|---|---|---|
| Primitive object | Fundamental string or brane | ZFA-closed event/history string |
| Background | Usually formulated on a spacetime background | Spacetime synthesized by events |
| Extra dimensions | Required for consistency in many formulations | Interpreted as orthogonal twist directions |
| Particle modes | Vibrations of strings | Stable spectra of ZFA histories |
| Vacuum landscape | Many possible compactifications | Possibilist sectors of ZFA closure |
| Gravity | Graviton mode / geometry | Event-synthesis delay and curvature |

In this sense, Universal Relativity does not merely reject string theory. It reinterprets string theory as an effective large-scale language for certain ZFA processes. The word **string** becomes literal again: a history string is an ordered expression of quantum-logical twists, and only closed strings of zero free action persist as physical events.
